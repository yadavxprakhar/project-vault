# Code Snippet Manager

## 📌 Overview
A developer tool for organizing, storing, and sharing code snippets with syntax highlighting, tagging, collections, and search functionality. Users can save frequently used code snippets, organize them into collections, add descriptions and tags, and share with the community. Features include syntax highlighting for 100+ languages, Markdown support, and version control.

## ✨ Features
- User authentication and personal snippet library
- Create, edit, and delete code snippets
- Syntax highlighting for 100+ programming languages
- Multi-file snippets (tabs for different files)
- Markdown descriptions with preview
- Tags and categories for organization
- Collections (folders) for grouping snippets
- Search snippets by title, description, tags, or code content
- Favorite/star snippets
- Public and private snippet visibility
- Share snippets via unique URLs
- Embed snippets in websites (iframe)
- Copy snippet to clipboard with one click
- Export snippets as Gist or files
- Import from GitHub Gists
- Snippet versioning and edit history
- Dark mode with multiple themes (VS Code, Dracula, etc.)
- Code execution for select languages (sandboxed)
- Responsive design for mobile code viewing

## 🛠️ Tech Stack
- **Frontend**: Next.js 14 with TypeScript or React + Vite
- **Syntax Highlighting**: Prism.js, Highlight.js, or Shiki
- **Code Editor**: Monaco Editor (VS Code) or CodeMirror
- **Backend**: Next.js API Routes or Express.js
- **Database**: PostgreSQL with Prisma or MongoDB
- **Authentication**: NextAuth.js or Clerk
- **Search**: Algolia, PostgreSQL full-text, or Fuse.js
- **Markdown**: react-markdown or MDX
- **Styling**: Tailwind CSS with shadcn/ui
- **Deployment**: Vercel or Railway
