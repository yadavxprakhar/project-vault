# Recipe Finder

## 📌 Overview
A recipe discovery application that allows users to search for recipes by ingredients, cuisine type, or dietary preferences. The app fetches data from a recipe API, displays search results with images and summaries, and shows detailed cooking instructions, ingredients, and nutritional information for selected recipes.

## ✨ Features
- Search recipes by keyword (e.g., "chicken", "pasta", "vegan")
- Filter by cuisine type (Italian, Mexican, Asian, etc.)
- Filter by dietary restrictions (vegetarian, vegan, gluten-free, etc.)
- Display recipe cards with image, title, and preparation time
- Detailed recipe view with full ingredient list
- Step-by-step cooking instructions
- Nutritional information (calories, protein, carbs, fats)
- Save favorite recipes to localStorage
- Random recipe suggestion button ("Surprise Me!")
- Ingredient checklist for shopping
- Serving size adjustment (scale ingredients)
- Recipe rating and review display
- Share recipe via social media or link
- Print-friendly recipe format
- Responsive grid layout for mobile and desktop

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla or React)
- **API**: Spoonacular API (free tier: 150 requests/day) or TheMealDB API
- **Alternative APIs**: Edamam Recipe API, Tasty API
- **Storage**: LocalStorage for favorite recipes
- **Styling**: CSS Grid for recipe cards, Flexbox, or Tailwind CSS
- **Icons**: Font Awesome for favorites, share, print icons
- **Optional**: React with useState/useEffect for API calls and state

## 📊 Difficulty Level
**Beginner** - Excellent for learning API integration, handling complex JSON responses, dynamic content rendering, and implementing search/filter functionality. Introduces working with image-heavy UIs and pagination.

## 🎯 Expected Outcomes
After completing this project, you will:
- Master API integration with authentication (API keys)
- Handle complex JSON data structures and nested objects
- Implement search functionality with query parameters
- Work with multiple filter criteria simultaneously
- Dynamically render image grids and cards
- Parse and display HTML content (instructions)
- Implement favorite/bookmark functionality
- Handle loading states and error messages
- Work with pagination or infinite scroll
- Optimize API calls to stay within free tier limits
