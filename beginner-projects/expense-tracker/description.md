# Expense Tracker

## 📌 Overview
A personal finance management application that helps users track income and expenses, categorize transactions, visualize spending patterns, and monitor their budget. The app provides insights into financial habits through charts and summaries, helping users make informed financial decisions.

## ✨ Features
- Add income and expense transactions with amount, description, and date
- Categorize transactions (food, transport, entertainment, bills, etc.)
- Edit and delete existing transactions
- Display current balance (total income - total expenses)
- Transaction history list with search and filter options
- Filter by date range (daily, weekly, monthly, custom)
- Category-wise expense breakdown with visual charts
- Budget setting for each category with alerts when exceeded
- Export transaction history to CSV
- Summary statistics (total income, total expenses, savings rate)
- Recurring transaction templates
- Data persistence using localStorage
- Responsive design for mobile and desktop
- Dark mode toggle

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla or React)
- **Storage**: LocalStorage for transaction data
- **Charts**: Chart.js or ApexCharts for data visualization
- **Styling**: CSS Grid/Flexbox or Tailwind CSS
- **Date Handling**: date-fns or Day.js
- **Icons**: Font Awesome or Material Icons
- **Optional**: React with Context API or Redux for state management

## 📊 Difficulty Level
**Beginner** - Excellent for learning form handling, data validation, array manipulation, and data visualization. Introduces basic financial logic and chart libraries while reinforcing CRUD operations.

## 🎯 Expected Outcomes
After completing this project, you will:
- Master form handling and input validation
- Work with date objects and date formatting
- Implement data filtering and search algorithms
- Use chart libraries for data visualization
- Practice array methods (reduce for totals, filter for categories)
- Handle number formatting (currency display)
- Implement budget tracking logic with conditional alerts
- Understand data export functionality (JSON to CSV)
- Build summary dashboards with calculated metrics
- Manage complex state with multiple data types
