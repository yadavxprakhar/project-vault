# Quiz App

## 📌 Overview
An interactive quiz application that allows users to test their knowledge on various topics. The app presents multiple-choice questions one at a time, tracks user answers, provides instant feedback, calculates scores, and displays results with performance insights. Perfect for learning, trivia, or educational assessments.

## ✨ Features
- Display multiple-choice questions one at a time
- Four answer options per question with radio buttons or clickable cards
- Next/Previous navigation between questions
- Question progress indicator (e.g., "Question 3 of 10")
- Timer for each question or entire quiz (optional countdown)
- Instant feedback on answer selection (correct/incorrect highlighting)
- Score tracking and final results page
- Percentage score calculation with pass/fail indicator
- Review answers after quiz completion
- Retry/restart quiz functionality
- Question categories (General Knowledge, Science, History, etc.)
- Difficulty levels (Easy, Medium, Hard)
- High score tracking with localStorage
- Responsive design for all devices
- Keyboard shortcuts (number keys 1-4 for answers, Enter for next)

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla or React)
- **Questions Data**: JSON file or JavaScript array of objects
- **Alternative**: Fetch questions from Open Trivia Database API
- **Storage**: LocalStorage for high scores and progress
- **Styling**: CSS Flexbox/Grid or Tailwind CSS
- **Timer**: JavaScript setInterval() or setTimeout()
- **Icons**: Font Awesome for checkmarks, X marks, timer icons
- **Optional**: React with useState/useEffect for state management

## 📊 Difficulty Level
**Beginner** - Excellent for learning conditional logic, array iteration, state management, and UI updates based on user interaction. Introduces timer functionality and score calculations without complex algorithms.

## 🎯 Expected Outcomes
After completing this project, you will:
- Master conditional logic and if/else statements
- Work with arrays of objects (question data structure)
- Implement timer functionality with setInterval/clearInterval
- Handle user input validation and state tracking
- Calculate percentages and display formatted results
- Dynamically update DOM based on application state
- Implement navigation between different app states (quiz/results)
- Use array methods like map, filter, and find
- Handle edge cases (unanswered questions, time expiration)
- Create responsive button and card layouts
