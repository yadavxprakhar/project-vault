# Habit Tracker

## 📌 Overview
A personal development application that helps users build and maintain positive habits through daily tracking, streak monitoring, and visual progress indicators. Users can create custom habits, check them off daily, view their consistency with calendar-style heatmaps, and receive motivation through streak counters and achievement milestones.

## ✨ Features
- Add custom habits with name and category
- Choose habit frequency (daily, weekdays, specific days, custom)
- Mark habits as complete for today with checkboxes
- Visual streak counter for each habit
- Calendar heatmap showing completion history
- Daily completion percentage and overview
- Weekly and monthly progress charts
- Habit categories (health, learning, productivity, fitness, etc.)
- Edit and delete habits
- Reset daily checkmarks at midnight
- Motivational messages for milestones (7-day streak, 30-day streak)
- Color coding for habit consistency (green = consistent, yellow = moderate, red = missed)
- Data persistence using localStorage
- Export/import habit data as JSON
- Responsive design for mobile and desktop

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla or React)
- **Storage**: LocalStorage for habit data and history
- **Charts**: Chart.js or ApexCharts for progress visualization
- **Calendar**: Custom calendar grid or FullCalendar.js
- **Dates**: date-fns or Day.js for date calculations
- **Styling**: CSS Grid for habit cards, or Tailwind CSS
- **Icons**: Font Awesome or Heroicons for categories

## 📊 Difficulty Level
**Beginner** - Perfect capstone project combining all beginner skills: CRUD operations, date manipulation, data visualization, and localStorage persistence. Teaches real-world productivity app development.

## 🎯 Expected Outcomes
After completing this project, you will:
- Master complex date calculations and calendar logic
- Implement streak algorithms (consecutive day counting)
- Build calendar heatmap visualizations from scratch
- Work with chart libraries for progress data
- Manage complex state with nested data structures
- Implement daily reset logic and timers
- Handle data migration and schema changes
- Build achievement/milestone systems
- Calculate statistics (completion rates, averages, trends)
- Export/import data functionality
