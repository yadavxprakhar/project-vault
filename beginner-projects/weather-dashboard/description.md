# Weather Dashboard

## 📌 Overview
A responsive weather application that displays current weather conditions and a 5-day forecast for any city worldwide. Users can search for cities and view temperature, humidity, wind speed, and weather conditions with dynamic icons and backgrounds that change based on weather.

## ✨ Features
- Search weather by city name with autocomplete suggestions
- Display current temperature, humidity, wind speed, and weather description
- 5-day weather forecast with daily highs/lows
- Dynamic weather icons based on conditions (sunny, rainy, cloudy, etc.)
- Temperature unit toggle (Celsius/Fahrenheit)
- Geolocation support to auto-detect user's location
- Recent searches history (last 5 cities)
- Responsive design for mobile and desktop
- Dynamic background color/image based on weather conditions
- UV index and air quality display
- "Feels like" temperature
- Sunrise and sunset times with visual indicators

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla or React)
- **API**: OpenWeatherMap API (free tier: 1000 calls/day)
- **Alternative APIs**: WeatherAPI, Tomorrow.io, Visual Crossing
- **Storage**: LocalStorage for search history and preferred units
- **Styling**: CSS Grid/Flexbox or Tailwind CSS
- **Icons**: OpenWeatherMap icons, Font Awesome weather icons, or custom SVGs
- **Geolocation**: Browser Geolocation API (built-in)

## 📊 Difficulty Level
**Beginner** - Perfect first API integration project. Focuses on async JavaScript, DOM manipulation, error handling, and responsive design fundamentals. The free API tier is sufficient for learning.

## 🎯 Expected Outcomes
After completing this project, you will:
- Understand how to work with REST APIs and fetch data
- Master async/await and Promise handling
- Handle API errors gracefully (invalid cities, network failures)
- Implement responsive design patterns with media queries
- Work with browser localStorage for data persistence
- Parse and display JSON data dynamically
- Use environment variables to secure API keys
- Implement loading states and skeleton screens
- Handle user input validation and sanitization
- Work with third-party icon libraries

## 🔥 Optional Enhancements
