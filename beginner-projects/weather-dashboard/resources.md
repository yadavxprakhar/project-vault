## 🔧 Tools & Libraries
- [OpenWeatherMap API](https://openweathermap.org/api) - Free tier available (1000 calls/day) - **RECOMMENDED**
- [WeatherAPI](https://www.weatherapi.com/) - Alternative weather API (generous free tier)
- [Tomorrow.io Weather API](https://www.tomorrow.io/weather-api/) - Professional weather data
- [Weather Icons](https://erikflowers.github.io/weather-icons/) - 222+ weather-themed icons
- [Axios](https://axios-http.com/) - Alternative to Fetch API for HTTP requests
- [dotenv](https://github.com/motdotla/dotenv) - Manage API keys securely (for Node.js projects)
